<h2 id="what-is-gnu-fm">What is GNU FM?</h2>

<p>GNU FM is the music community platform created in 2009 for the GNU Project.
It implements the Audioscrobbler protocol, powers <a href="https://libre.fm">libre.fm</a>
and supports federation between many Auidoscrobbler servers, including Last.fm.</p>

<h2 id="where-can-i-join-gnu-fm">Where can I join GNU FM?</h2>

<p>The main GNU FM instance is <a href="https://libre.fm">libre.fm</a> .</p>

<p>The front page says (among other things):</p>

<p>&ldquo;Share your listening habits from your phone, computer or tablet, and find
 similiar artists.
All the music on Libre.fm is freely licensed for you to stream, download and
 remix.</p>

<p>The Libre.fm community is a mixture of music superfans and artists, where
artists can share their music for everyone to enjoy.</p>

<p>We believe we represent a good deal for artists: we only promote artists who
choose to share their music with the world.&rdquo;</p>

<h2 id="how-can-i-run-my-own-instance-of-gnu-fm">How can I run my own instance of GNU FM?</h2>

<p>You can check out the source code for GNU FM from its git repository -
<a href="https://git.gnu.io/gnu/gnu-fm/">https://git.gnu.io/gnu/gnu-fm/</a></p>

<p>The source code includes
<a href="https://git.gnu.io/gnu/gnu-fm/blob/master/gnufm_install.txt">installation instructions</a>.</p>

<h2 id="i-found-a-bug-how-can-i-report-it-or-get-help">I found a bug, how can I report it or get help?</h2>

<p>You can report a bug using the issue tracker at
<a href="https://git.gnu.io/gnu/gnu-fm/issues">https://git.gnu.io/gnu/gnu-fm/issues</a> .</p>

<p>If you use IRC you can contact us on  the GNU fm IRC channel at
<a href="https://webchat.freenode.net/?channels=#libre.fm">#libre.fm</a>.</p>

<h2 id="who-makes-gnu-fm">Who makes GNU FM?</h2>

<p>The maintainer of the GNU FM project is Rob Myers.</p>

<p>The founders of the GNU FM project were Matt Lee, Michael Sheldon, Clint Adams,
and Rob Myers.</p>

<p>Thank you to Bytemark, who provide the servers that run libre.fm .</p>
